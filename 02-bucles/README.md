# Clase 2 – Estructuras Repetitivas (Ciclos) en PSeInt y C++


## Objetivo
En esta clase aprenderás a automatizar procesos mediante estructuras repetitivas:

* Comprender qué es un ciclo o bucle
* Utilizar estructuras repetitivas (`do-while`, `while`,  `for`)
* Controlar la ejecución mediante condiciones
* Aplicar ciclos en problemas reales
* Traducir algoritmos iterativos de PSeInt a C++



## ¿Qué aprenderás?
Al finalizar esta clase serás capaz de:

* Diseñar algoritmos que repiten procesos de manera controlada
* Implementar ciclos en PSeInt
* Escribir programas en C++ usando `do-while`, `while` y `for`
* Resolver problemas como sumatorias, validaciones y menús interactivos



¿Cómo usar este material?
Sigue este orden:


1. Leer los archivos en teoria/. Analizar ejemplos paso a paso, ejecutar en PSeInt y luego en C++
2. Resolver los ejercicios en ejercicios/

## Contenido de la clase

### 1. Introducción a los ciclos

* ¿Qué es una estructura repetitiva?
* Cuándo usar ciclos en lugar de repetir código manualmente





### 2. Ciclo do-while

* Ejecución garantizada al menos una vez
* Ejemplo: menú de opciones


### 3. Ciclo while

* Diferencia con do-while
* Estructura básica
* Condición de entrada
* Ejemplos: conteo, validación de datos



### 4. Ciclo for

* Uso con contadores
* Inicialización, condición y actualización
* Ejemplos: tablas, sumatorias, recorridos



### 5. Aplicaciones prácticas


* Suma de números
* Promedios
* Validación de entrada (ej: pedir nota hasta que sea válida)
* Menú interactivo (calculadora básica)



### 6. Traducción a C++
* Pasar algoritmos de PSeInt a código real
* Uso correcto de llaves {}
* Control de flujo



## Errores comunes

* Generar bucles infinitos (condición mal definida)
* Olvidar actualizar la variable de control
* Usar mal = en lugar de == en condiciones
* No identificar cuándo usar `do-while`, `while`  o `for`
* No probar casos límite (ej: valores inválidos)
