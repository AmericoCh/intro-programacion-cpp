# Clase 1 – Introducción a la Programación (PSeInt, C++)

## Objetivo

En esta clase veremos los fundamentos de la programación desde cero:

* Comprender qué es un algoritmo
* Usar variables y entrada/salida
* Aplicar estructuras secuenciales
* Tomar decisiones con condicionales (`if-else`, `switch`)
* Traducir algoritmos de pseudocódigo a C++

---

## ¿Qué aprenderás?

Al finalizar esta clase serás capaz de:

* Escribir algoritmos básicos en PSeInt
* Crear programas simples en C++
* Entender la lógica detrás de un programa
---

## ¿Cómo usar este material?

Sigue este orden:

1. Leer los archivos en `teoria/`. Revisar y ejecutar los ejemplos.

2. Resolver los ejercicios en `ejercicios/`

## Requisitos

Para esta clase necesitas:

entre otros)

## Errores comunes

* Intentar memorizar código sin entender la lógica
* Olvidar declarar variables
* Confundir `=` (asignación) con `==` (igualdad)
* No probar el programa paso a paso
